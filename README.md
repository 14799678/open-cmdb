Django rest framework + vue 的CMDB项目


## 环境

- Python 3.6
    - Django 2.0
    - Django Rest Framework 3.8
    
- Vue.js 2.9
    - iview 3.0
    - iview-admin 2.0

## 功能

- CMDB资源管理
- 报表展示
- web ssh
